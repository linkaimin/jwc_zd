## 1.1. tips

- 用户类别：
  - 1 ： 管理员
  - 2 ： 教师
- code ：
  - 0 ： 正常
  - 1 ： 添加 message 字段展示相关信息。（例如：删除，更新，插入。会有 “删除成功/失败” 信息）
  - 2 ： 登录失败

- data ： 返回数据
- url ：host + url
  - example ：
    - host ：<http://localhost/jwc>
    - url ：/login
    - url : <http://localhost/jwc/login>
